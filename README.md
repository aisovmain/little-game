# 🎯 Угадай число — Консольная C# игра



**Угадай число от 1 до 100** за минимальное количество попыток! Компьютер даёт подсказки "больше/меньше". Игра **бесконечная** — новые раунды после победы.

## ✨ Фичи
- ✅ **Русский ASCII** "УГАДАЙ ЧИСЛО"
- ✅ **Цветные подсказки** (зелёный/красный)
- ✅ **Бесконечные раунды**
- ✅ **Счётчик попыток**
- ✅ **ESC** — выход в любой момент
- ✅ **200 строк**, 0 зависимостей
